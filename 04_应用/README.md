# 应用开发（RAG 与 Agent）

本目录包含大语言模型应用开发相关的学习资料和代码实现。

## 学习内容

### RAG（检索增强生成）
- Embedding 技术
- 向量数据库（MongoDB、Chroma、Pinecone、Milvus）
- Rerank 算法
- 混合检索策略

### Agent（智能体）
- 工具调用（Tool Calling）
- 任务规划与反思
- 多 Agent 协作

### 开发框架
- LangChain
- LlamaIndex
- AutoGen
- Semantic Kernel
